# IoT Time Series Forecasting & Anomaly Detection

## Overview
This project implements an end-to-end IoT system that simulates sensor data,
applies time-series machine learning, and detects anomalies using statistical
methods and LSTM neural networks.

## Features
- Real-time IoT sensor simulation
- Time-series data ingestion
- LSTM-based forecasting
- Statistical anomaly detection
- Interactive Streamlit dashboard

## Tech Stack
- Python
- Pandas, NumPy
- TensorFlow (LSTM)
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
python iot/sensor_simulator.py
python ml/train_lstm.py
streamlit run app.py
```
